# LDCF-Model-by-PyTorch

This repository implements the LDCF model using PyTorch.

The original paper is:

> Location-Aware Deep Collaborative Filtering for Service Recommendation

## Introduction

This project is a PyTorch implementation of the LDCF model.

The model focuses on location-aware service recommendation by integrating:
- Collaborative Filtering
- Location Information
- Deep Neural Networks
